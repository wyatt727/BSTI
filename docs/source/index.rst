Welcome to BSTI Nessus Documentation
==================================

BSTI Nessus is a modern tool for converting Nessus CSV files to Plextrac format and uploading findings to Plextrac.

Features
--------

* Convert Nessus CSV files to Plextrac format
* Upload findings to Plextrac
* Configuration wizard for easy setup
* Secure credential management
* Progress reporting for long-running operations
* Parallel processing for performance

Installation
-----------

To install BSTI Nessus, run:

.. code-block:: bash

   pip install bsti-nessus

Or from source:

.. code-block:: bash

   git clone https://github.com/bsti-security/bsti-nessus.git
   cd bsti-nessus
   pip install -e .
